# HoloPulse Fit Visual Assets

This folder contains placeholder visual assets for the HoloPulse Fit Android app package.

Included groups:

1. branding: LumaStride logo, HoloPulse Fit app icon, and SVG source.
2. ui_reference: landing page, camera workout, and Ghost Trainer layout references.
3. exercise_pose_reference: squat, jumping jack, push-up, and gesture pose references.
4. icons_used: camera, workout, history, settings, replay, timer, energy, gesture, streak, and calories icons.
5. animations: small Lottie style JSON placeholders for progress and Aura Energy states.
6. palette: color palette JSON and image swatch.
7. sample_data: sample workout history and motion path JSON for review.

These assets are safe placeholders and may be replaced by final custom graphics during development.
